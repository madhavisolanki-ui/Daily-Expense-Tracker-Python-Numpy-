# Daily-Expense-Tracker-Python-Numpy-
This project is a simple command-line based Daily Expense Tracker developed using Python. It takes daily expense inputs from the user and performs calculations such as total expense, average expense, and highest expense using NumPy. The project also provides a warning message if the average expense exceeds a predefined limit.

**Technologies Used**
1. Python
2. NumPy


**Features**
1. User input for number of days
2. Stores daily expenses
3. Calculates:
   -> Total Expense
   -> Average Expense
4. Highest Expense
5. Displays warning if average expense is high
6. Uses NumPy for efficient calculations

   
**Learning Outcomes**
1. Understanding of Python input/output
2. Use of NumPy arrays
3. Basic data analysis concepts
4. Conditional logic in Python
